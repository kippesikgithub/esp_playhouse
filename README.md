## Needed Parts
- Esp32
- 5v waterpump
- ws2812 leds
- Solar panel
- 18650 Lithium Ion Battery cells
- Solar/Battery Power manager (BMS) https://www.tinytronics.nl/shop/nl/power/bms-en-laders/zonneladers/dfrobot-zonne-energie-manager-5v
